### Backend 
   
* Installer les packages
   ```sh
   npm install
   ```
* Lancer le serveur
   ```sh
   npm run start
   ```
   
* Le message suivant doit s'afficher 
   ```sh
    > groupomania-backend@1.1.0 start
    > node server.js

    Listening on port 3000
    Connexion à MongoDB réussie !
   ```

### Frontend

* Installer les packages
   ```sh
   npm install
   ```
   
* Lancer le serveur
   ```sh
   npm run dev
   ```
   
* Le message suivant doit s'afficher 
   ```sh
  VITE v3.1.0  
  ➜  Local:   votre adresse
   ```

#### Le site Gropomania est désormais fonctionnel !
